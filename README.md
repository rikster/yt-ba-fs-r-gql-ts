## Full Stack Reddit Server App
### Ground up
- React
- TypeScript
- GraphQL
- URQL/Apollo
- Node.js
- PostgreSQL
- MikroORM/TypeORM
- Redis
- Next.js
- TypeGraphQL
- Chakra
### Setup
1. npm init -y
2. yarn add -D @types/node typescript 
   <br/>(DEV, types for node)
3. yarn add -D ts-node
   <br/>(this is slow)
4. npx tsconfig.json
   (or create/copy manually)
